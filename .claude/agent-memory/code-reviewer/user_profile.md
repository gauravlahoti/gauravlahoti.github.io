Gaurav Lahoti is the portfolio owner and primary developer. He is a Senior Cloud & AI-Native Architect at Deloitte. The portfolio is a personal project showcasing his work, written entirely in vanilla HTML/CSS/JS with no framework or build toolchain. He uses a spec-driven workflow: every feature is documented in `.claude/specs/` before implementation.

He expects reviews to be thorough, prioritized by severity, and actionable. He values:
- Strict adherence to CSS variable conventions (no hardcoded hex)
- Mobile UX correctness against DoD items
- Semantic HTML and accessibility correctness
- No dead markup or dead code shipped to production
